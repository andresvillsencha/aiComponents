Smart Fill Component

This example demonstrates how to use the Smart Fill Component within an Ext JS form to automatically analize the contents of a field and fill out a form.

The user can copy-and-paste any text and paste it in the field, and it will automatically analize it's contents and fill out all the other form.

The component connects to a backend NodeJS Middleware service powered by OpenAI or Anthropic, which interprets the user's input and returns a structured JSON containing the fields data.
